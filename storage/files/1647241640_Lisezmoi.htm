Logiciels Adobe® Acrobat® Pro DC et Adobe® Acrobat® Standard DC

 

Bienvenue dans ce fichier Lisez-moi Adobe® Acrobat® Pro DC et Adobe® Acrobat®  Standard DC, qui contient des informations essentielles sur les produits Adobe Acrobat. Pour les éléments de ce fichier Lisez-moi qui s’appliquent à tous les produits, aucun nom de produit spécifique n’est indiqué. Pour toute information concernant le support technique du logiciel Adobe Acrobat DC, rendez-vous sur http://www.adobe.com/go/acrobat_support_fr

 

Configuration requise

 

La configuration minimale requise pour exécuter Acrobat Pro DC et Acrobat Standard DC for Windows® est la suivante :

 

·         processeur 1,5 GHz ;

·         Microsoft® Windows Server® 2008 R2 (32 bits et 64 bits) ou 2012 R2 (64 bits), Windows 7 (32 bits et 64 bits), Windows 8 (32 bits et 64 bits) ou Windows 8.1 (32 bits et 64 bits) ;

·         1 Go de RAM ;

·         2,5 Go d’espace disponible sur le disque dur si vous effectuez l’installation à partir d’un support DVD, ou 4,5 Go d’espace disponible sur le disque dur si vous effectuez l’installation à partir d’un package en téléchargement (Remarque : Le programme d’installation crée des fichiers temporaires pendant l’installation ; l’espace libre nécessaire sur le disque dur à ce moment est donc supérieur à celui indiqué.) ;

·         résolution d’écran de 1024 x 768 ;

·         navigateur Internet Explorer® 8, 9, 10 (Windows 8 minimum) ou 11, ou Firefox (ESR) ;

·         lecteur DVD-ROM, si vous effectuez l’installation à partir d’un DVD ;

·         accélération matérielle de la vidéo (facultatif).

 

Instructions d’installation

 

Pour installer le logiciel ou pour mettre à niveau une version antérieure, procédez comme suit. Si vous effectuez une mise à niveau, vous pourrez être invité à forunir une preuve de propriété.

Pour installer Adobe Acrobat DC sur un système d’exploitation Windows

 

1. Redémarrez Windows et désactivez tout logiciel antivirus.

 

2. Connectez votre ordinateur à Internet.

 

3. Débutez l’installation :

  • Si vous avez acheté Acrobat sur un support DVD, insérez le DVD.
  • Si vous avez acheté Acrobat en téléchargement, téléchargez votre package à l’emplacement souhaité sur votre disque dur.

 

Adobe vous recommande également de sauvegarder votre package en téléchargement, de préférence sur un DVD, avant d’installer le logiciel. Double-cliquez sur le fichier .exe pour le décompresser :

 

un dossier <system drive>:\users\<user name>\Desktop\Adobe Acrobat DC

ou

un dossier <system drive>:\users\<user name>\Desktop\Adobe Acrobat 2015

 

 est créé sur le bureau.

 

Lorsque vous êtes prêt à commencer l’installation, ouvrez ce dossier et double-cliquez sur le fichier setup.exe.

 

L’écran d’installation s’affiche dans la même langue que celle du système d’exploitation de votre ordinateur. Il vous fournit des informations sur l’enregistrement du numéro de série ou l’installation en mode d’évaluation, la propriété des PDF et le programme de mise à jour. Assurez-vous que la langue de votre système d’exploitation est définie sur Anglais ou sur la langue dans laquelle vous avez acheté le logiciel.

 

4. Saisissez votre numéro de série ou sélectionnez l’installation en mode d’évaluation. Le numéro de série des licences achetées individuellement est indiqué à l’extérieur du boîtier du DVD, ou sur la page de reçu pour les logiciels à diffusion par voie électronique dont les achats sont effectués en ligne, par exemple sur l’Adobe Store. Dans le cas des licences individuelles de logiciels à diffusion par voie électronique, le numéro de série est également indiqué dans l’historique de votre commande en ligne sur http://www.adobe.com/fr. Dans le cas des licences achetées sur le site Internet Adobe Licensing, le numéro de série est également indiqué sur votre compte sur http://licensing.adobe.com.

 

5. Pour effectuer une installation personnalisée, par exemple pour modifier l’emplacement du dossier de l’application, cliquez sur le bouton « Personnaliser » et sélectionnez le dossier dans lequel vous souhaitez installer l’application.

 

Cliquez sur « Installer » pour lancer l’installation.

 

6. Si certains processus en cours d’exécution gênent l’installation, le programme d’installation vous invite à les fermer avant de poursuivre l’installation. Fermez ces processus, puis cliquez sur « Continuer » dans l’acran d’installation pour poursuivre l’installation.

 

7. Lorsque la boîte de dialogie « Installation terminée » s’affiche, cliquez sur « Terminer », ou cliquez sur « Lancer maintenant » pour lancer Acrobat.

 

Première utilisation et instructions relatives à la saisie des identifiants Adobe

 

1. Au lancement de l’application, la boîte de dialogue Connexion requise s’affiche. Cliquez sur Me connecter maintenant et saisissez un identifiant Adobe valide dans l’écran Connexion.

 

2. Si vous ne possédez pas d’identifiant Adobe, procédez comme vous y êtes invité pour en créer un.

 

3. Cliquez sur « Connexion ».

 

4. Cliquez sur « Accepter » dans l’écran Contrat de licence utilisateur final. Vous devriez à présent pouvoir utiliser Acrobat.

 

5. Si vous avez installé une version d’essai, un écran Essai de 30 jours contenant un bouton Démarrer l’essai s’affiche une fois que vous avez accepté le Contrat de licence utilisateur final. Cliquez sur ce bouton pour démarrer l’essai. Un écran Essai s’affichera par la suite une fois par jour au lancement du logiciel pour indiquer le nombre de jours d’essai restant. Une fois cette période d’essai de 30 jours terminée, vous ne pourrez utiliser le logiciel que pour afficher des fichiers PDF et effectuer des tâches basiques, par exemple saisir des commentaires, remplir des formulaires et apposer des signatures.

Pour désinstaller Acrobat DC sur un système d’exploitation Windows

 

1. Ouvrez le Panneau de configuration, puis sélectionnez Programmes et fonctionnalités.

 

2. Dans la liste des applications qui s’affiche, mettez Adobe Acrobat DC en surbrillance et cliquez sur le bouton Désinstaller.

 

3. Une boîte de dialogue s’affiche pour vous demander de confirmer la désinstallation d’Acrobat. Pour désinstaller le produit, sélectionnez Oui.
