Adobe® Acrobat® Pro DC-Software und Adobe® Acrobat® Standard DC-Software

 

Dies ist die Liesmich-Datei zu Adobe® Acrobat® Pro DC und Adobe® Acrobat®  Standard DC, die wichtige Informationen zu Adobe Acrobat-Produkten enthält. Im Hinblick auf Informationen in der Liesmich-Datei, die für alle Produkte gelten, wird der Produktname nicht genannt. Informationen zum Produkt-Support für Adobe Acrobat DC erhalten Sie unter http://www.adobe.com/go/acrobat_support_de.

 

Systemanforderungen

 

Für Acrobat Pro DC und Acrobat Standard DC für Windows® sind folgende Systemvoraussetzungen erforderlich:

 

·         Prozessor mit 1,5 GHz

·         Microsoft® Windows Server® 2008 R2 (32-Bit und 64-Bit) oder 2012 R2 (64-Bit); Windows 7 (32-Bit und 64-Bit); Windows 8 (32-Bit und 64-Bit); Windows 8.1 (32-Bit und 64-Bit)

·         1 GB RAM

·         2,5 GB freier Festplattenspeicher bei Installation von einer DVD und 4,5 GB freier Festplattenspeicher bei Installation mit einem Downloadpaket. Da das Installationsprogramm bei der Installation temporäre Dateien erstellt, ist der zu diesem Zeitpunkt tatsächlich erforderliche Festplattenspeicher größer als angegeben.

·         Bildschirmauflösung von 1024 x 768

·         Internet Explorer® 8, 9, 10 (mindestens Windows 8), 11; Firefox (ESR)

·         DVD-ROM-Laufwerk bei Installation von einer DVD

·         Optionale Videohardwarebeschleunigung

 

Installationsanweisungen

 

Installieren Sie neue Software oder führen Sie ein Upgrade von einer älteren Version gemäß den folgenden Anweisungen durch. Falls Sie ein Upgrade installieren, werden Sie möglicherweise aufgefordert, einen Eigentumsnachweis zu erbringen.

So installieren Sie Adobe Acrobat DC unter Windows

 

1. Starten Sie Windows neu und deaktivieren Sie etwaige Virenschutzprogramme.

 

2. Verbinden Sie den Computer mit dem Internet.

 

3. Beginnen Sie mit der Installation:

  • Wenn Sie Acrobat auf einer DVD erworben haben, legen Sie die DVD ein.
  • Haben Sie Acrobat als Download erworben, laden Sie das Paket auf Ihre Festplatte herunter.

 

Adobe empfiehlt vor der Installation der Software, das Downloadpaket zu sichern, und zwar bevorzugt auf einer DVD. Nachdem Sie zum Entpacken des Pakets auf die Datei mit der Endung „.exe“ geklickt haben, geschieht Folgendes:

 

<Systemlaufwerk>:\Benutzer\<Benutzername>\Desktop\Adobe Acrobat DC

oder

<Systemlaufwerk>:\Benutzer\<Benutzername>\Desktop\Adobe Acrobat 2015

 

 wird auf dem Desktop erstellt.

 

Um mit der Installation zu beginnen, öffnen Sie den entsprechenden Ordner und doppelklicken Sie auf die Datei „Setup.exe“.

 

Der Installationsbildschirm wird in der Sprache Ihres Betriebssystems angezeigt. Er enthält Informationen zur Seriennummer/Installation der Testversion, zur PDF-Eigentümerschaft und zum Aktualisierungsprogramm. Stellen Sie sicher, dass als Sprache des Betriebssystems entweder Englisch oder die erworbene Sprache eingestellt ist.

 

4. Geben Sie die Seriennummer ein oder installieren Sie das Produkt im Testmodus. Die Seriennummer einzeln lizenzierter Produkte steht außen auf der DVD-Hülle. Wurde das Produkt als elektronische Softwareverteilung (ESD) online erworben, z. B. im Adobe Store, entnehmen Sie die Seriennummer der Seite, auf der der Kauf bestätigt wurde. Bei individuell lizenzierten EDS-Käufen wird die Seriennummer auch im Online-Bestellverlauf unter http://www.adobe.com/de vermerkt. Die Seriennummer bei über die Lizenzierungswebsite von Adobe getätigten Käufen steht im entsprechenden Datensatz unter http://licensing.adobe.com.

 

5. Für eine benutzerdefinierte Installation, z. B. wenn Sie den Pfad der Anwendung ändern möchten, klicken Sie auf „Anpassen“ und geben Sie den Ordner an, in dem die Anwendung installiert werden soll.

 

Klicken Sie auf „Installieren“. Die Installation wird nun fortgesetzt.

 

6. Sollten Prozesse ausgeführt werden, die mit der Installation in Konflikt stehen, werden Sie vom Installationsprogramm aufgefordert, diese anzuhalten, bevor die Installation fortgesetzt werden kann. Beenden Sie die Prozesse und klicken Sie in das Installationsprogramm, um die Installation fortzusetzen.

 

7. Zuletzt wird das Dialogfeld „Installation beendet“ angezeigt. Klicken Sie auf „Fertigstellen“ oder auf „Jetzt starten“, um Acrobat zu starten.

 

Anweisungen zum ersten Starten und zur Adobe-ID-Eingabe

 

1. Nach dem Starten der Anwendung wird das Dialogfeld „Anmeldung erforderlich“ angezeigt. Klicken Sie auf „Jetzt anmelden“ und geben Sie im Bildschirm zum Anmelden eine gültige Adobe-ID ein.

 

2. Wenn Sie keine Adobe-ID haben, befolgen Sie die Anweisungen, um eine Adobe-ID zu erstellen.

 

3. Klicken Sie auf „Anmelden“.

 

4. Akzeptieren Sie nun die Endanwenderlizenzvereinbarung. Acrobat kann jetzt verwendet werden.

 

5. Wenn Sie eine Testversion installiert haben, wird der Bildschirm der 30-tägigen Testversion mit der Schaltfläche „Testversion starten“ angezeigt, nachdem Sie die Endanwenderlizenzvereinbarung akzeptiert haben. Klicken Sie auf die Schaltfläche „Testversion starten“, um den Testzeitraum zu starten. Nach dem Starten erscheint einmal täglich der Bildschirm „Testversion“, um die Anzahl der verbliebenen Tage der Testversion anzuzeigen. Nach 30 Tagen endet die Testversion und Sie können die Software nur noch zum Anzeigen von PDF-Dateien und für grundlegende Aufgaben wie Kommentieren, Ausfüllen von Formularen und Unterzeichnen verwenden.

So deinstallieren Sie Acrobat DC unter Windows

 

1. Öffnen Sie die Systemsteuerung und wählen Sie „Programme und Funktionen“ aus.

 

2. Wählen Sie in der Anwendungsliste den Eintrag „Adobe Acrobat DC“ aus und klicken Sie auf „Deinstallieren“.

 

3. In einem Dialogfeld werden Sie aufgefordert, die Deinstallation von Acrobat zu bestätigen. Klicken Sie zur Deinstallation der Anwendung auf „Ja“.
